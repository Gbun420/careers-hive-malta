# Careers.mt Brand Assets

## Core Colors
- **Primary Navy**: `#1B2331` (Backgrounds, text)
- **Secondary Coral**: `#FF5444` (CTAs, interactive elements)
- **Premium Gold**: `#D5AA16` (Featured status, trust badges)

## Logo System
1. **Logo Horizontal** (`logo-horizontal.svg`): Primary logo for headers.
2. **Logo Stacked** (`logo-stacked.svg`): Secondary logo for small spaces or center-aligned layouts.
3. **Mark** (`mark.svg`): Icon-only version for profile pictures or social media.
4. **Favicon** (`favicon.svg`): Optimized 32x32 mark.

## Usage Rules
- **Safe Area**: Maintain at least 10px of clear space around the horizontal logo.
- **Minimum Size**: Horizontal logo should not be rendered smaller than 120px wide.
- **Accessibility**: Never place Coral text on a Navy background without verifying contrast. Use Coral for buttons and Navy for high-contrast text.
